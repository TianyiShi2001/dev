knitr Supports Many More Languages Than You Might Expect

Although it’s easy to insert R/Python/SQL/Bash/Rcpp/D3/Stan code chunks using RStudio’s GUI, many more languages are supported. The following commmand lists all languages available:

names(knitr::knit_engines$get())

Here I wrote a simple function in Rcpp that finds the nth fibonacci number.

#include <Rcpp.h>

// [[Rcpp::export]]
int fibonacci(const int x) {
  if (x == 0 || x == 1) return(x);
  return (fibonacci(x-1) + fibonacci(x - 2));
}

Then I can call it in an R code chunk:

fibonacci(10L)
## [1] 55
console.log({result: "hello world"})
